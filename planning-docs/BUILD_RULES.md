# SmartCut Builder v1.0 – Rules Set for Cabinet Boxes / Furniture Carcasses

## 6.1 Base Gaps & Clearances (All units in inches)

| Gap                          | Typical Placement                                           | Minimum | Notes                      |
| ---------------------------- | ----------------------------------------------------------- | ------- | -------------------------- |
| **Back‑to‑Back (Rear)**      | Between rear wall & rear panel of drawer or door            | 0.125   | For proper hinge clearance |
| **Side‑to‑Side**             | Between adjacent drawers/doors                              | 0.125   | Avoids contact             |
| **Front‑to‑Front**           | Front wall vs front panel of drawer/door                    | 0.125   | Allows smooth pull         |
| **Top‑to‑Top**               | Between top face & uppermost panel (e.g., top of a cabinet) | 0.125   | For latch clearance        |
| **Bottom‑to‑Bottom**         | Between bottom wall & lowest panel                          | 0.125   | For foot clearance         |
| **Inside Clearance (Joint)** | Between joined faces (e.g., tongue‑to‑groove)               | 0.020   | Allows glue line thickness |
| **Door‑to‑Door**             | Overlap of door panels                                      | 0.125   | For hinges                 |
| **Drawer‑to‑Drawer**         | Overlap                                                     | 0.125   | For smooth glides          |

## 6.2 Joinery & Cut Adjustments

| Joinery             | Cut‑out Adjustment                  | Description                       |
| ------------------- | ----------------------------------- | --------------------------------- |
| **Tongue & Groove** | Subtract `0.020` from adjacent face | Provides a 0.02″ glue line        |
| **Mitered Corner**  | Add `0.020` to each side            | Allows for glue & slight over‑lap |
| **Chamfer**         | Add `0.020` to adjacent faces       | For edge protection               |
| **S-Shape**         | Subtract `0.020` from both ends     | For seamless connection           |
