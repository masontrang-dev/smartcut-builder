## Detailed Milestone Breakdown

### MVP Phase (Nov 2025 – May 2026)

#### Month 1-2: Foundation

- Project setup & infrastructure
- Database schema implementation
- Authentication system
- Basic CRUD APIs

#### Month 3-4: Core Features

- Box geometry engine
- Drawer/door calculator
- Rule engine for gaps & clearances
- Material library

#### Month 5-6: AI & Export

- OpenAI integration
- Smart defaults wizard
- PDF/CSV cut list generation
- Assembly instructions generator

### V2 Phase (May 2026 – Nov 2026)

#### Month 7-8: Visualization

- Three.js integration
- 3D preview component
- Interactive model viewer

#### Month 9-10: Optimization

- Cut sheet optimizer
- Hardware diagrams
- Joinery visualizations

#### Month 11-12: Advanced Export

- DXF export
- SketchUp export
- Enhanced PDF layouts

### V3 Phase (Nov 2026 – Nov 2027)

#### Quarter 1: AI Enhancement

- AI Design Coach
- Structural validation
- Cost estimation engine

#### Quarter 2: Community

- Template marketplace
- User ratings & reviews
- Community sharing

#### Quarter 3: AR/VR

- AR room preview (WebXR)
- Mobile AR support
- Tool recommendation engine

#### Quarter 4: Voice & Polish

- Voice-guided assistant
- Performance optimization
- Beta testing & refinement
